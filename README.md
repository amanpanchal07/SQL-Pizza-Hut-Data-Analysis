# SQL-Pizza-Hut-Data-Analysis
SQL project based on Pizza Hut sales data analysis to identify top-selling pizzas, revenue trends, customer ordering patterns, and business insights using SQL queries.

# Pizza Hut Sales Analysis

This SQL project analyzes Pizza Hut sales data to find business insights using SQL queries.

## Analysis Included
- Top-selling pizzas
- Total revenue analysis
- Order trends
- Category-wise sales analysis
- Customer ordering patterns

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- SUM()
- AVG()
- COUNT()
- JOINS

## Tools Used
- MySQL
- SQL
- GitHub
